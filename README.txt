------------------------------------------------------
QUICK START
------------------------------------------------------
1. Open docs/index.html in your browser — done!
   No build tools or npm required.

2. Replace "VORTEX" with your team name throughout
   index.html (Ctrl+F / Cmd+F → Replace All)

3. Set your next match date in script.js:
   const targetDate = new Date('YYYY-MM-DDTHH:MM:SS');

4. Customize brand colors in style.css under :root {}

5. Deploy by uploading the 3 files in docs/ to any
   static host (GitHub Pages, Netlify, Vercel, etc.)

KEY FEATURES
------------------------------------------------------
- Real-time countdown timer (1 line to configure)
- 5 player cards with 3D tilt hover effect
- Match schedule with status badges (WIN / UPCOMING)
- Animated hero stats (counter animation on scroll)
- Mouse-following cursor glow
- Twitch stream mockup UI
- Social feed posts section
- Sponsors grid (title + regular tiers)
- Tryout application form
- Full mobile responsiveness + hamburger nav

------------------------------------------------------
CUSTOMIZATION TIPS
------------------------------------------------------
- Brand color: change --orange in style.css :root
  (also update --orange-dim and --orange-border)
- Countdown date: 1 line in script.js
- Player cards: edit player-card blocks in index.html
- Add/remove matches: duplicate/delete .match-row divs
